# Simon-Game
The Simon Game, programmed with C for an ATtiny1626 microcontroller hardware device.

**Functionality includes:**

Pseudo random number sequence generation for Simon's noises and display illumination

Displays user's score based on amount of correct guesses through display illumination

Adjustable playback delay using a potentiometer

Ability to increase or decrease playback frequencies

Interaction with hardware switches **OR**

Transmission through UART for convenience for primary functionalities

Transmission through UART to reset the Simon Game, or begin with a new seed for the pseudo random number generator

Displays five highest scores of all players
